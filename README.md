# 📄 ATS Resume Analyzer

## 📌 Overview
This project implements an Applicant Tracking System (ATS) Resume Analyzer that evaluates how well a resume matches a given job description. It uses Natural Language Processing (NLP) techniques to extract and compare text, helping users understand their resume's relevance.

---

## 🚀 Features
- Resume and job description comparison  
- TF-IDF based text vectorization  
- Cosine similarity scoring  
- Keyword extraction and matching  
- Simple interface for input and analysis  

---

## 🛠️ Tech Stack
- Python  
- Scikit-learn  
- Pandas  
- NLP techniques (TF-IDF, text preprocessing)  

---

## ⚙️ How It Works

1. **Input**
   - Resume text  
   - Job description  

2. **Processing**
   - Text cleaning and preprocessing  
   - TF-IDF vectorization  

3. **Matching**
   - Compute cosine similarity  
   - Extract important keywords  

4. **Output**
   - Similarity score  
   - Matching insights
